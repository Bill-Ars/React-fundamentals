# React Fundamentals 🚀

Projeto desenvolvido durante meus estudos de React.js, com foco nos fundamentos da biblioteca e na criação de componentes reutilizáveis.

##Sobre o projeto

Este projeto reúne exercícios e componentes criados para praticar os principais conceitos do React:

- Componentes
- JSX
- Props
- State (`useState`)
- Eventos
- Renderização condicional
- Organização de componentes
- Estilização com CSS

O objetivo é construir uma base sólida em React antes de avançar para conceitos mais avançados como React Router, Context API e consumo de APIs.

---

## Tecnologias utilizadas

- React.js
- Vite
- JavaScript (ES6+)
- CSS3
- HTML5
- Git e GitHub
